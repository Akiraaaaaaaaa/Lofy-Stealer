##  Configuração Manuaç:
 
Primeiro cole e salve seu webhook em `"WEBHOOK HERE"` em Lofy.py

Se você usar o ofuscador, ele será indetectável.

se você tiver um erro durante a instalação, tente `pip install -r requirements.txt`

Agora você precisa usar o pyinstaller para converter o arquivo python em exe.

Abra o CMD e digite `pip install auto_py_to_exe`

E depois de instalado `python -m auto_py_to_exe`

Procurar arquivo Selecionar `Um arquivo é baseado no Windows (ocultar o console)`
